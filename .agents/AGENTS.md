# Project Rules

- Όταν καταχωρείς/δημοσιεύεις μια νέα έκδοση (version) και το ιστορικό της (release notes/changelog):
  - Πρέπει πάντα η έκδοση να ορίζεται στο τρίτο επίπεδο (π.χ. 1.1.1, 1.2.1).
  - Πρέπει πάντα να καταγράφεται ρητά εάν η αλλαγή αφορά τον **Server**, τον **Client** ή και τα δύο (**Server & Client**).
